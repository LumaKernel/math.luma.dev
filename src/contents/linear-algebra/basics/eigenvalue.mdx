## <Q w="eigenvalue"k d/>

<Q w="square-matrix"/>$A \in \Complex^{n \times n}$に対し、$A\mathbf x = \lambda \mathbf x$を$\lambda \in \Complex$、$\mathbf x \in \Complex^n$（ただし$\mathbf x\neq \mathbf 0$）が満たすとき、

- $\lambda$を<Q w="eigenvalue"/>
- $\mathbf x$を<Q w="eigenvector"/>

とそれぞれ呼ぶ。

なお、そのような$\mathbf x$は定数倍しても<Q w="eigenvector"/>のままである。

<Q w="eigenvalue"/>と<Q w="eigenvector"/>すべてを求める問題を<Q w="eigenvalue"k/>問題という。

## <Q w="eigenvalue"k/>問題の計算方法

$A \mathbf x = \lambda I \mathbf x$より$(A - \lambda \idmat) \mathbf x = \mathbf 0$となる。これは<Q w="homo-system"/>だ。

$A - \lambda \idmat$が<Q w="invertible-mat"/>であるなら、$(A - \lambda \idmat)^{-1}$を左から乗じて$\mathbf x = \mathbf 0$となるため、<Q w="invertible-mat"/>ではない。

よって、$\lambda$は$|A - \lambda I| = 0$を満たす。この$\lambda$に関する<Q w="equation"/>を<Q w="eigenvalue-eq"d/>と呼ぶ。

逆に、$\lambda$が$|A - \lambda I| = 0$を満たしたのなら$(A-\lambda I) = \mathbf 0$は$\mathbf x \neq 0$以外の<Q w="sol"/>を持つため、<Q w="equation"/>を調べれば、すべての<Q w="eigenvalue"/>を調べたことになる。
