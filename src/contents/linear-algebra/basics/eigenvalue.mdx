## 固有値

正方行列$A \in \Complex^{n \times n}$に対し、$A\mathbf x = \lambda \mathbf x$を$\lambda \in \Complex$、$\mathbf x \in \Complex^n$（ただし$\mathbf x\neq \mathbf 0$）が満たすとき、

- $\lambda$を固有値
- $\mathbf x$を固有値ベクトル

とそれぞれ呼ぶ。

なお、そのような$\mathbf x$は定数倍しても固有値ベクトルのままである。

固有値と固有値ベクトルすべてを求める問題を固有値問題という。

## 固有値問題の計算方法

$A \mathbf x = \lambda I \mathbf x$より$(A - \lambda \idmat) \mathbf x = \mathbf 0$となる。これは斉次方程式だ。

$A - \lambda \idmat$が正則行列であるなら、$(A - \lambda \idmat)^{-1}$を左から乗じて$\mathbf x = \mathbf 0$となるため、正則行列ではない。

よって、$\lambda$は$|A - \lambda I| = 0$を満たす。この$\lambda$に関する方程式を固有値方程式と呼ぶ。

逆に、$\lambda$が$|A - \lambda I| = 0$を満たしたのなら$(A-\lambda I) = \mathbf 0$は$\mathbf x \neq 0$以外の解を持つため、方程式を調べれば、すべての固有値を調べたことになる。
