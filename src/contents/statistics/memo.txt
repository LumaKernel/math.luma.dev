
### **基本的な関係**

* **ベルヌーイ分布** `(Bernoulli)`
    * → `n回の独立試行の和` → **二項分布** `(Binomial)`

* **二項分布** `(Binomial)`
    * → `n=1` → **ベルヌーイ分布** `(Bernoulli)`
    * → `np=λ (定数), n→∞, p→0` → **ポアソン分布** `(Poisson)`
    * → `n→∞` → **正規分布** `(Normal)` (ド・モアブル–ラプラスの定理)

* **ポアソン分布** `(Poisson)`
    * → `λ→∞` → **正規分布** `(Normal)`
    * → `最初の事象発生までの待ち時間` → **指数分布** `(Exponential)`
    * → `k回目の事象発生までの待ち時間` → **ガンマ分布** `(Gamma)` / **アーラン分布** `(Erlang)`

* **幾何分布** `(Geometric)`
    * → `無記憶性を連続時間へ拡張` → **指数分布** `(Exponential)`
    * → **負の二項分布** `(Negative Binomial)` の `r=1` の特殊ケース

* **負の二項分布** `(Negative Binomial)`
    * → `r=1` → **幾何分布** `(Geometric)`

* **超幾何分布** `(Hypergeometric)`
    * → `母集団サイズN→∞` → **二項分布** `(Binomial)`

---

### **連続分布間の関係**

* **正規分布** `(Normal)`
    * → `μ=0, σ²=1 に標準化` → **標準正規分布** `(Standard Normal)`
    * → `標準正規分布に従う確率変数の2乗` → **自由度1のカイ二乗分布** `(Chi-squared)`
    * → `確率変数を指数関数の肩に乗せる (e^X)` → **対数正規分布** `(Log-normal)`
    * → `独立な2つの標準正規分布の比 (X/Y)` → **コーシー分布** `(Cauchy)`

* **ガンマ分布** `(Gamma)`
    * → `形状パラメータα=1` → **指数分布** `(Exponential)`
    * → `形状パラメータαが整数n` → **アーラン分布** `(Erlang)`
    * → `形状パラメータα=n/2, 尺度パラメータβ=2` → **自由度nのカイ二乗分布** `(Chi-squared)`
    * → `形状パラメータα→∞` → **正規分布** `(Normal)`

* **ベータ分布** `(Beta)`
    * → `α=1, β=1` → **連続一様分布** `(Continuous Uniform)`

* **指数分布** `(Exponential)`
    * → `独立な指数分布の和` → **ガンマ分布** `(Gamma)` / **アーラン分布** `(Erlang)`

---

### **正規分布から派生する分布（統計的検定で多用）**

* **カイ二乗分布** `(Chi-squared)`
    * `独立な標準正規分布の2乗和` → **カイ二乗分布** `(Chi-squared)`
    * → `自由度n→∞` → **正規分布** `(Normal)`

* **t分布** `(Student's t)`
    * `標準正規分布 / √ (カイ二乗分布/自由度)` → **t分布** `(Student's t)`
    * → `自由度n→∞` → **標準正規分布** `(Standard Normal)`
    * → `自由度n=1` → **コーシー分布** `(Cauchy)`

* **F分布** `(F-distribution)`
    * `(カイ二乗分布1/自由度1) / (カイ二乗分布2/自由度2)` → **F分布** `(F-distribution)`








### 1変量分布の関係（基本）

まず、基本的な1変量分布の関係をおさらいします。

* **二項分布** → `np=λ, n→∞` → **ポアソン分布**
* **二項分布** → `n→∞` → **正規分布**
* **ポアソン分布** → `λ→∞` → **正規分布**
* **ポアソン分布** → `事象の待ち時間` → **指数分布**・**ガンマ分布**
* **ガンマ分布** → `形状パラメータα=1` → **指数分布**
* **ガンマ分布** → `特殊なパラメータ` → **カイ二乗分布**
* **t分布** → `自由度→∞` → **標準正規分布**

---

### 多変量分布への拡張と関係

1変量分布を多変量（多次元）に一般化することで、新たな関係性が生まれます。

* **二項分布** `(Binomial)`
    * → `試行結果のカテゴリ数をk個に一般化` → **多項分布** `(Multinomial)`

* **多項分布** `(Multinomial)`
    * → `カテゴリ数 k=2` → **二項分布** `(Binomial)`
    * → `特定のカテゴリに着目した周辺分布` → **二項分布** `(Binomial)`
    * → `試行回数 n→∞` → **多変量正規分布** `(Multivariate Normal)`

* **ベータ分布** `(Beta)`
    * → `次元をkに一般化` → **ディリクレ分布** `(Dirichlet)`

* **ディリクレ分布** `(Dirichlet)`
    * → `次元 k=2` → **ベータ分布** `(Beta)`
    * → `独立なガンマ分布の変数変換` → **ディリクレ分布** は、複数の独立なガンマ分布に従う確率変数を、その和で割ることで導出できます。

* **正規分布** `(Normal)`
    * → `確率変数をベクトルに拡張` → **多変量正規分布** `(Multivariate Normal)`

* **多変量正規分布** `(Multivariate Normal)`
    * → `任意の1変数を取り出した周辺分布` → **正規分布** `(Normal)`
    * → `一部の変数を固定したときの条件付き分布` → **正規分布** `(Normal)`
    * → `中心からのマハラノビス距離の2乗` → **カイ二乗分布** `(Chi-squared)`

* **カイ二乗分布** `(Chi-squared)`
    * → `多変量への一般化 (共分散行列の分布)` → **ウィシャート分布** `(Wishart)`

* **ウィシャート分布** `(Wishart)`
    * `多変量正規分布からサンプリングしたベクトルを用いて作られる平方和行列の分布` として導出されます。
    * → `次元 p=1` → **カイ二乗分布** `(Chi-squared)` または **ガンマ分布** `(Gamma)`

---

### ベイズ統計における共役事前分布の関係

ベイズ統計学では、特定の尤度関数に対して、事後分布が事前分布と同じ形の分布になる「共役」な関係が重要です。これは直接的な導出とは異なりますが、分布間の非常に強い結びつきを示します。

| 尤度関数（データが従う分布） | 共役事前分布（パラメータが従う分布） |
| :--- | :--- |
| **ベルヌーイ分布 / 二項分布** | **ベータ分布** `(Beta)` |
| **多項分布** `(Multinomial)` | **ディリクレ分布** `(Dirichlet)` |
| **ポアソン分布** `(Poisson)` | **ガンマ分布** `(Gamma)` |
| **指数分布** `(Exponential)` | **ガンマ分布** `(Gamma)` |
| **正規分布** `(平均が未知)` | **正規分布** `(Normal)` |
| **正規分布** `(分散が未知)` | **逆ガンマ分布** `(Inverse Gamma)` |
| **多変量正規分布** `(平均ベクトルが未知)`| **多変量正規分布** `(Multivariate Normal)`|
| **多変量正規分布** `(共分散行列が未知)`| **逆ウィシャート分布** `(Inverse Wishart)`|

---

### その他の重要な関係性（分布族）

* **指数型分布族** `(Exponential Family)`
    * 正規分布、二項分布、ポアソン分布、ガンマ分布、ベータ分布、ディリクレ分布など、これまで挙げた多くの主要な分布が属する大きなクラスです。この族に属する分布は数学的に共通の形式を持ち、一般化線形モデル（GLM）などの統計モデルで統一的に扱うことができます。

* **安定分布** `(Stable Distribution)`
    * 独立同分布に従う確率変数の和を考えたとき、その和が元の分布と同じ形状になる（スケールや位置が異なるだけ）という性質を持つ分布族です。
    * **正規分布**、**コーシー分布**、**レヴィ分布**は安定分布の特殊なケースです。中心極限定理の一般化と考えることもできます。
